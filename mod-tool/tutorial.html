📖 Mod Editor 使用教程

本教程将指导你如何使用 TrayBuddy Mod Editor 创建和编辑角色 Mod 包。

简介

TrayBuddy Mod Editor 是一个可视化的 Mod 编辑工具，让你更直观地管理角色的各种数据。

📋
基础信息
配置 Mod ID、版本、作者、角色渲染参数、Mod 数据面板等
🎭
状态和触发
管理动作状态、对话分支、事件响应逻辑以及数据计数器
💬
气泡样式
自定义对话气泡样式，支持实时预览效果
🖼️
动画
管理序列帧动画和静态图片配置
📝
文本管理
多语言文本编辑，支持简易 Markdown
🔊
音频管理
管理多语言语音资源配置

环境要求

⚠️ 浏览器要求

请使用 Chrome 86+ 或 Edge 86+ 浏览器，其他浏览器暂不支持文件夹读写功能。

推荐运行方式：双击 mod-tool/打开-mod编辑器.bat 启动本地服务器（会自动打开浏览器），然后访问 http://127.0.0.1:4174/

快速开始

打开现有 Mod 文件夹

点击「打开 Mod」下拉菜单

在工具栏点击「📂 打开 Mod」按钮，选择「📂 打开文件夹」

选择 Mod 文件夹

在弹出的选择器中，选择一个 Mod 文件夹（如 ema）

授予访问权限

浏览器会询问是否允许读写该文件夹，点击「允许」

打开 .tbuddy 文件

.tbuddy 是 TrayBuddy 专用的 Mod 包格式，本质上是一个 ZIP 压缩包。点击「📦 打开 .tbuddy」选择文件后自动加载。

创建新 Mod

点击「➕ 新建 Mod」，填写 Mod ID、名称和作者，系统会创建一个包含默认配置的新 Mod。

📋 基础信息配置

在「基础信息」标签页中配置 Mod 的核心参数。

Mod 信息

字段说明
Mod ID唯一标识符，只允许英文和数字
版本号Mod 版本
作者创作者名称
默认音频语言找不到对应语言音频时的备选
默认文本语言找不到对应语言文本时的备选

角色配置

配置角色渲染相关参数（manifest.character）。

字段说明
z_offset角色画布的渲染层级偏移
canvas_fit_preference角色 Canvas 显示适配：short(短边优先)/long(长边优先)/legacy(旧版按高度缩放)

边框配置

配置边框开关、边框动画与层级（manifest.border）。

字段说明
enable是否启用边框
anima边框使用的动画名称（从动画资源中选择）
z_offset边框画布的渲染层级偏移

Mod 数据面板

可在角色窗口左上角显示自定义数据（如好感度）。配置 show_mod_data_panel 开关和 mod_data_default_int 默认值。

预览图与图标

预览图支持 PNG/JPG/JPEG/WebP 格式，图标支持 ICO/PNG 格式。

🎭 状态和触发

重要状态

系统内置的关键状态：

状态名称持久说明
idle✅默认空闲状态，必须存在
silence✅静默/免打扰状态
silence_start❌进入免打扰时播放一次，然后跳转到 silence
silence_end❌退出免打扰时播放一次，然后跳转到 idle
dragging✅拖动中状态（持续）
drag_start❌开始拖动时播放一次，然后跳转到 dragging
drag_end❌结束拖动时播放一次，然后跳转到 idle（或后续由系统进入 music）
music✅音乐播放状态
music_start❌开始听歌时播放一次，然后跳转到 music
music_end❌停止听歌时播放一次，然后跳转到 idle
birthday❌生日事件状态（通常播放一次）
firstday❌首次启动事件状态（通常播放一次）

状态字段

字段说明
状态名称唯一标识符
是否持久持久状态循环播放
关联动画要播放的动画
关联音频音频资源名称
关联文本显示为气泡对话
优先级数值越大优先级越高
下一状态播放完成后跳转
触发间隔定时触发检查间隔（秒），0 表示不启用
触发概率每次检查的触发概率（编辑器以百分比显示，保存到 manifest 为 0-1）
可触发子状态在编辑器内以“状态 + 权重”列表配置（权重越大越容易被选中）

对话分支

对话分支允许用户通过点击选项来决定对话走向。每个分支包含显示文本和跳转目标状态。

“显示气泡”(branch_show_bubble) 用于控制分支选项是否以气泡形式展示；该选项位于「对话分支」折叠区内。

触发器配置

触发器定义事件与状态的关联。常用事件（支持 keydown:<Key>）：

事件名触发时机
click用户点击角色
login程序启动
animation_drag_start开始拖动 Animation 窗口
animation_drag_end结束拖动 Animation 窗口
keydown:<Key>按键触发（例如：keydown:Space）
birthday生日事件
firstday首次启动事件
login_silence启动后进入静默/无声状态
music_start音乐开始播放
music_end音乐停止播放

数据计数器

状态激活时对 Mod 数据执行操作：set(设置)、add(加)、sub(减)、mul(乘)、div(除)。

💬 气泡样式配置

可视化配置对话气泡外观，支持实时预览。

  • 气泡主体：背景、边框、圆角、阴影、背景滤镜
  • 文字样式：颜色、字体大小、行高、字体
  • 尾巴样式：大小、颜色、阴影
  • 装饰文字：上方/下方装饰字符
  • 分支按钮：正常/悬浮/点击状态样式

🖼️ 动画管理

动画资源分为序列帧动画(sequence.json)和杂图(img.json)两类。

字段说明
名称资源名称，供状态引用
图片路径相对于 asset/ 目录
是否序列帧是否为多帧动画
帧间隔每帧显示时间(秒)
帧尺寸单帧宽高(像素)
帧数横向/纵向帧数量
需要反向播放开启后会以乒乓方式播放（默认关闭，可在高级选项中设置）
原始反向图片序列本身为反向时可开启，用于纠正方向
偏移X/Y 偏移（像素），用于微调渲染位置
⚠️ 注意

动画图片需要手动放置到 asset/sequence/ 或 asset/img/ 目录。

📝 多语言文本

配置多语言的角色信息和对话内容。

角色信息 (info.json)

语言ID、语言名称、角色名称、角色描述

对话文本 (speech.json)

名称、内容(支持Markdown)、持续时间(秒)

Markdown 语法

  • **文本** → 加粗
  • [文本](链接) → 超链接
  • \n → 换行

🔊 多语言音频

配置多语言的语音资源。音频文件支持 .wav 格式，需要手动放置到对应目录。

💾 保存与导出

保存到文件夹

点击「💾 保存」（或「📦 导出」→「📂 到文件夹」），选择目标文件夹后写入。当前版本每次保存都会弹出文件夹选择器。

导出为 .tbuddy

点击「📦 导出」→「📦 到 .tbuddy」生成压缩包（包内会以 Mod ID 作为根目录）。

保存/导出的文件

  • manifest.json
  • bubble_style.json（开启气泡样式时生成）
  • asset/sequence.json, asset/img.json
  • text/[lang]/info.json, text/[lang]/speech.json（开启文本对话时生成）
  • audio/[lang]/speech.json（开启音频对话时生成）
  • preview.(png/jpg/jpeg/webp), icon.ico

📁 Mod 目录结构

mods/my_mod/
├── manifest.json            # Mod 主配置文件
├── preview.png              # 预览图
├── icon.ico                 # 图标
├── bubble_style.json        # [可选] 气泡样式
├── asset/
│   ├── img.json, sequence.json
│   ├── img/, sequence/
├── audio/[lang]/
│   ├── speech.json, speech/
└── text/[lang]/
    ├── info.json, speech.json

❓ 常见问题

Q: 为什么无法打开文件夹？

A: 请使用 Chrome 86+ 或 Edge 86+ 浏览器。

Q: 动画不显示？

A: 检查图片是否在正确目录，路径和帧配置是否正确。

Q: 如何添加新语言？

A: 在「文本管理」或「音频管理」中点击「➕ 添加语言」。

Q: 预览图支持哪些格式？

A: 支持 PNG、JPG、JPEG、WebP 四种格式。

Q: 为什么提示 i18n 资源加载失败 / file:// 被拦截？

A: 请不要用 file:// 直接打开页面。双击 mod-tool/打开-mod编辑器.bat 启动本地服务器，再用浏览器访问 http://127.0.0.1:4174/。
